# Credit-Card-Fraud-Detection


This project focuses on detecting fraudulent credit card transactions using machine learning models. Since fraudulent transactions are very rare compared to normal transactions, the dataset is highly imbalanced, making fraud detection a challenging problem. The goal is to accurately identify fraud while minimizing false positives.

* Key Insight

Random Forest performed the best because it reduces overfitting and handles imbalanced data more effectively.

Fraud detection systems require high recall to ensure suspicious cases are not missed.

This project demonstrates how machine learning can be used to:
